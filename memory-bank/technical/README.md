## Technology Stack

1. **Frontend**
   - Next.js with React
   - TypeScript
   - Tailwind CSS
   - React Query
   - Zustand

2. **Backend**
   - Python with FastAPI
   - Microservices architecture
   - OpenTelemetry integration
   - AI capabilities

3. **Databases**
   - PostgreSQL (Primary)
   - MongoDB (Content)
   - Pinecone (Vector)
   - Elasticsearch (Search)
   - Redis (Caching)

4. **Infrastructure**
   - Docker containers
   - Kubernetes orchestration
   - LGTM Stack monitoring
   - CI/CD pipeline
